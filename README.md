# Nikola Rubic Portfolio

Static bilingual portfolio website for Nikola Rubic.

## Local Preview

Open `index.html` directly in a browser or serve the folder with a simple static server.

## GitHub Pages

Upload the complete folder contents to a GitHub repository and publish from the repository root. The site uses relative paths and does not require a build step.

## SEO Deployment Notes

- Submit `https://nrubic-hub.github.io/portfoliowebsite-nr/sitemap.xml` in Google Search Console and Bing Webmaster Tools.
- Add Google Search Console or Bing verification meta tags in the `<head>` after the services provide the verification IDs.
- Add GA4 or Google Tag Manager only after a real measurement ID/container ID is available. The site currently ships without tracking cookies.
- Keep canonical URLs aligned with the final GitHub Pages URL or future custom domain.
